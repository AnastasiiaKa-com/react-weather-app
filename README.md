# Weather App

Modern weather application built with React, TypeScript and Vite.

## Features

- Search weather by city
- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- 5-day forecast
- Clean glassmorphism UI

## Tech Stack

- React
- TypeScript
- Vite
- CSS Modules
- OpenWeather API

## Installation

Clone the repository:

git clone https://github.com/your-username/weather-app.git

Install dependencies:

npm install

Run development server:

npm run dev
